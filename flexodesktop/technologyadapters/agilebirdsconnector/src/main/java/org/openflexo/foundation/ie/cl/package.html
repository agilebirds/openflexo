This package provides classes used by the Flexo Component Library.

The {@link FlexoComponentLibrary} is a tree of {@link FlexoComponentFolder} where each {@link FlexoComponentFolder} may contains sub-folders and/or some {@link ComponentDefinition}.
A {@link ComponentDefinition} is a reference to a {@link org.openflexo.foundation.IEWOComponent}. A {@link ComponentDefinition} can be used as a representation of his {@link org.openflexo.foundation.IEWOComponent} and doesn't require to load the whole {@link org.openflexo.foundation.IEWOComponent} in memory. @author Benoit Mangez
