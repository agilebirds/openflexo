Represents the whole data model for the project.
  • Here are stored and abstractally represented all the data sources usable in the context of current project.
  • Those data may come from various sources, such as external libraries, internal code, internal or external databases, thesaurus, webservices, etc...
  • All concepts are formelly represented by entities. An entity generally map a Java class (if Java language is used).
  • All entities sharing same implementation type (source code, database, webservice, etc...) are grouped into a repository.
