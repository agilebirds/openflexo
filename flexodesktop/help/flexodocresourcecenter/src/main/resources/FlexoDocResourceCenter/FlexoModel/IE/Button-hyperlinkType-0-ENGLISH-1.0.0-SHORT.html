Type of Hyperlink associated with this button. This type can be
  • FlexoAction: action that participate to the workflow and a Petri token move. Form values are submitted and saved in the current process instance.
  • DisplayAction: that does not participate to the workflow and that usually jump to another leve of the process. Form are usually not submitted ans the form values are not saved in the current process instance.
  • Search: associated with search functionality. Search field is submitted.
  • URL: move to the associated url.
  • Mail-To: creat a mailto HTML hyperlink.
  • ClientSideScript: add a javascript to the button.
