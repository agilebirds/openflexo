The Palette contains all the graphical elements that can be dragged in the central panel to compose a screen.
The palette has four tabs:

  1. Basic tab with the basic Graphicul User Interface (GUI) widgets, i.e. graphical elements.
  2. Icon tab with predefined images that can be dragged and dropped in the central panel.
  3. Button tab with predefined buttons that can be dragged and dropped in the central panel.
  4. Widget tab with user defined widgets.
For exporting a Widget to the Palette, select the composed elements you want to transformed into a widget, right click on it and select "Export Widget to Palette" action.

Widgets are defined in xml documents that can be found in the .prj file to be reused in other .prj's.
