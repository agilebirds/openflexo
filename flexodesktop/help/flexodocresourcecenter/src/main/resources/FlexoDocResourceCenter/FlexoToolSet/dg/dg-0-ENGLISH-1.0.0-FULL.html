Openflexo Documentation Generator provides an extensive support for editing, configuring and generating documentation targeting various audiences (localized contents) and formats (e.g. business, functional or technical specifications, operating procedures).
The Openflexo Documentation Generator provides a powerful tool for collaborative documentation edition, ensuring that the documentation is always in sync with the modelling. Also provides round trip facilities, allowing people to share documentation in textual versions and for some format re-injection in the model.
