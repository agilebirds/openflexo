This view helps you to generate and write on the disk LaTeX and screenshots. Afterwards, you can easily generate a PDF document based on what is on your disk.

There are 3 buttons available:
  1. Synchronize: regenerates LaTeX and screenshots but does not write them on the disk.
  2. Synchronize and write: regenerates LaTeX and screenshots but does not write them on the disk.
  3. Generate PDF: generates the PDF based on what can be found on the disk.
